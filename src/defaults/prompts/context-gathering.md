Method:

Early stop criteria:
- You can identify exact actions needed.
- Results converge on one area or approach.

Escalate:
- If signals conflict or scope is unclear, run one refined search batch, then proceed.

Depth:
- Only trace information directly relevant to the current task.

Loop:
- Search -> plan -> execute.
- Search again only if validation fails or new unknowns appear.
- Prefer acting over more searching.
